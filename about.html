About Project

The purpose of this project is to find and analyze news based on the entered search word. News are loaded using newsapi.org, with indication of date and media source.

On the Analytics page there is a graph showing how the number of mentions has been distributed during every day of the last week, as well as total statistics of the number of mentions in news and in news titles only.

The About Project page also contains a slider with my GitHub commits to this project.

The coding has been performed as per BEM-methodology. BEM makes the site flexible and scalable. It allows easy and prompt adding of new pages and functionality which leads to easier site support.

The cards are fetched asynchronously from the server. It means that the site continues to operate and wait for the response from server. Quite convenient.

Webpack technology has been used during project build. The code is optimized, hence the site works quicker. It is also supports old browsers.
